（一）Nodejs的安装

  1. 根据您的电脑系统去nodejs官网下载nodejs的安装包。
    注意：是msi格式的文件，且要下载8.4以上版本的。
  2. 安装步骤很简单，只需要一直下一步，msi文件是专门为微软系统提供的安装包，所以安装完成后无需进行任何配置。
  3. 安装完成后，打开cmd,输入 node –v ,出现版本号则说明安装成功。

（三）运行前端代码

  1. 打开cmd,使用cd命令转到前端代码所在目录，并输入 npm install,按回车。
  2. 等待几分钟后，安装依赖完成。然后输入 npm run dev。在浏览器输入 http://localhost:8080 则可以查看前端界面。输入 http://localhost:8080/#/admin/userList 可以查看全局管理员的页面。 注意：生产阶段不需要运行前端代码！既“3.运行前端代码”本步骤不需要执行。

（四）MongoDB安装与配置

  1. 根据您的电脑系统去MongoDB官网下载MongoDB的安装包。
    注意：是msi格式的文件。
  2. 安装步骤很简单，只需要一直下一步，msi文件是专门为微软系统提供的安装包，环境变量已自动配置好，所以安装完成后无需进行任何配置。
  3. 在安装文件所在盘符创建一个任意名的文件夹,用于数据存放目录。
  4. 打开cmd,使用cd命令进入安装文件夹 ,输入mongod –dbpath d:\mongo。这里的“mongo”为刚刚创建的用于存放数据的文件夹。
  5. 出现如图所示，则说明安装成功。此时在浏览器中输入“127.0.0.1:27017”,则出现 注意：此cmd不能关闭，需保持开启。
    （mongodb的安装与配置，网上有多种方法，在此仅提供最快速的方法。例如，如何将mongoDB设置为Windows服务。在此也不赘述。）

（五）运行本系统

  1. 打开cmd,使用cd命令进入到项目根目录，执行“npm install”,此步骤为安装依赖。
  2. 安装依赖完毕后，根目录会出现“node_modules”文件夹。然后输入“npm run start”。
